Step 1: 사진 촬영

사용자가 얼굴 또는 체형 사진을 업로드합니다. PhotoAnalysis.jsx에서 파일 선택 또는 카메라 캡처를 지원합니다.

Step 2: CLIP 임베딩

OpenCLIP 모델로 이미지를 512차원 벡터로 변환합니다. clip_embedder.py에서 처리하며, 이전 사진과의 유사도 비교가 가능합니다.

Step 3: 건강 추정

photo_analyzer.py가 체형 변화, 피부 상태, 안색 등을 분석하여 건강 지표를 추정합니다. BMI 변화 추이도 추적합니다.

Step 4: Top-5 조언

분석 결과를 기반으로 개선 우선순위 상위 5개 항목을 추천합니다. 식단·운동·수면·스트레스 등 크로스 도메인 조언을 포함합니다.

Step 5: 12주 로드맵

Top-5 조언을 기반으로 RoadmapTimeline.jsx에 12주 맞춤형 개선 로드맵을 생성합니다. 주차별 목표와 체크포인트를 포함합니다.

음성 명령 시나리오 (4가지)

🍳 “오늘 저녁 뭐 먹을까?”

Intent: food / recommend

→ food_agent → 냉장고 재료 기반 레시피 추천 → 영양 분석 → 음성으로 안내

🏃 “러닝 30분 기록해줘”

Intent: exercise / log

→ exercise_agent → 운동 기록 저장 → 칼로리 소모 계산 → 주간 달성률 업데이트

🏥 “건강검진 결과 분석해줘”

Intent: health / analyze

→ health_agent → 검진 데이터 파싱 → 위험 지표 경고 → 개선 조언 생성

🎯 보상 함수 핵심 로직

R(s, a, t) = w1·r_taste + w2·r_health + w3·r_fitness + w4·r_mood + w5·r_habit
             - penalty_night - penalty_risk - penalty_skip + bonus_photo_goal

Penalties (감점)

🍜 야식 라면 (23시) -5

→ 수면 -35% → 운동 -20% → 체중 목표 +2일 지연

🍗 튀김 치킨 -4

→ 콜레스테롤 위험 +12%

🌫 미세먼지 러닝 (PM10 ≥ 76) -4

→ 활성산소 증가 → 혈액 청정도 하락

Bonuses (가점)

🍳 에어프라이어 수용 +2

→ 칼로리 -40%

🎸 기타 연주 30분 +2

→ 스트레스 -15% → 폭식 충동 -40%

🏥 건강검진 이행 +4

→ 불안감 감소 → 전체 안정

📸 사진 목표 50% 달성 +3

→ 시각적 동기부여 → 습관 강화

⚡ 빠른 시작 (로컬)

# 1. 프로젝트 클론
git clone <repo-url> && cd MOMA/lifesync-ai

# 2. 환경변수 설정
cp .env.example .env
# .env 파일 편집하여 API 키 입력

# 3. 백엔드 실행
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
uvicorn backend.app.main:app --reload --port 8000

# 4. 프론트엔드 실행 (새 터미널)
cd frontend
npm install
npm run dev

# 5. Docker로 전체 실행 (선택)
docker-compose up --build
